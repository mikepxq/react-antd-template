# react-antd-template

> 这是一个简单的后端管理模板。它使用 vite 快速编译(启动<1s),使用eslint prettier lint commitlint 统一代码格式,使用 react-router concent axios mock 实现主要功能。

## Build step
```
# clone

# 进入项目目录

# 安装依赖
npm install

# 启动
npm run dev

# 生成模式
npm run build
```
## 其他

```
# 预览
npm run preview 

# 代码格式化及自动修复
npm run lint:fix
```
